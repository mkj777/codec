## Codec
Codec is a Lightweight Game Library Client. Hopefully a Place for everything you could need.

<img width="1776" height="1054" alt="525727318-c87af48a-e254-443e-bade-9651a7d26382" src="https://github.com/user-attachments/assets/c071c71d-2ef0-4fc8-a7b4-0c7dde763e80" />

### Features
Scanning your PC for all your Games  
Viewing Details about your Games, and launching them through Codec  
Finding Game Folders and adding Launch Scripts  
Launching Games directly through Codec

### Installation
Download the latest release "Codec_Installer_x.x.x.exe" from the [Releases](https://github.com/mkj777/codec-releases/releases) section on GitHub  

### Requirements
Windows 10 or later  
.NET 9.0 Runtime  

### Development
Codec is a C# WinUI 3 Project using .NET 9.0  
using Inno Setup for compiling the installer  
thanks to [@Deadlock-too](https://github.com/Deadlock-too) for the npm package [howlongtobeat-ts](https://github.com/Deadlock-too/howlongtobeat-ts) to use the howlongtobeat API
