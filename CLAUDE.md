This Project is a of the WinUI 3 Framework.  
Its a Game Library, that should be minimalistic and performant. 
It scans the PC for Games, manual adding also possible.
Used should always be MVVM patterns first, if possible/available and best practice Software Architecture.
Most important is that the Codebase is clean, readable and maintainable for a Solo Dev, no extensive Documentation needed, but a well structured Codebase is a must.
Lean towards using less complicated code, stay simple.
